Let's Talk CSS Selectors!

There are three main types of selectors.

  1. Elements
    • * is the "Universal" selector. It selects everything. You typically use these to remove element default styles which makes it easier for you to modify each elements location and style.
    • html element is the next most broad selector, followed by body.
    • You select this items in you css style sheet by simply typing their name. For example, you can style the body element by typing body{background-color: #d3d3d3d3}
  2. Classes
    • Classes allow you to style individual elements.
    • In CSS classes are prefixed by a dot. Where with elements, you simply type the element name, with classes, you type .class-name.
    • Separate multi-word class names with a " - ". If you add a space, it will be treated like you are creating two separate classes.
    • You can use a class as many times as you'd like.
  3. ids
    • IDs are similar to classes. Instead of using a dot as a prefix, you use hash symbol.
    • Unlike classes, you can only use an ID a single time in an HTML document.
    • You select this items in you css style sheet by simply typing their name. For example, you can style the body element by typing body{background-color: #d3d3d3d3}
